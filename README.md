# 🌾 KisanBazaar – India's Farmers Marketplace

A modern, dark-themed web marketplace where farmers can list their produce, livestock, fertilizers, and agricultural equipment — and buyers can browse, search, and contact sellers directly.

**🔗 Live Site:** _[Add your GitHub Pages URL here after deploying]_

---

## ✨ Features

- 🥬 **Farm Produce** listings — vegetables, grains, fruits
- 🐄 **Livestock** — cows, buffaloes, goats, poultry
- ♻️ **Fertilizers & Dung** — compost, vermicompost, cow dung cakes
- 🚜 **Tools & Equipment** — tractor rentals, tillers, harvesters
- 🔍 **Search & filter** by category, location, keyword
- 📞 **Contact modal** with WhatsApp and Call links
- 📝 **Post Listing** form for sellers
- 📱 Fully **responsive** design

## 🚀 Tech Stack

- Pure HTML5, CSS3, Vanilla JavaScript
- No frameworks, no dependencies — works offline too
- Hosted on **GitHub Pages**

## 📂 Project Structure

```
kisanbazaar/
├── index.html       # Main page
├── index.css        # Design system + all styles
├── app.js           # Listings data, search, filter, modals
└── hero.png         # Hero banner image
```

## 🛠️ Run Locally

Just open `index.html` in any browser — no build step needed.

Or serve with Python:
```bash
python3 -m http.server 8765
```
Then visit `http://localhost:8765`
